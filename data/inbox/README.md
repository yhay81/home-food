# Inbox

受け取ったレシート画像・冷蔵庫画像・料理写真などの原本を保存する場所です。**家計簿の証憑になるレシート画像は、必ずここへ原本をコピーします**（外食のテキスト報告など、そもそも原本画像が無い入力は対象外）。

推奨パス:

```text
data/inbox/YYYY-MM-DD/<元のファイル名>
```

原本を保存したら、`data/sources/catalog.yml` の `raw_location` に **この workspace の相対パス**（例 `data/inbox/2026-06-28/xxxx.jpg`）を記録します。`~/.openclaw/inbound/...` のような workspace 外の絶対パスは、このエージェントから到達できずバックアップもされないため書きません。原本画像が無い入力（テキスト報告のみ等）は `raw_location: null` で構いません。

このディレクトリは Git 追跡対象で、push すると private remote にバックアップされます。個人情報を含むため、極端に機微なものはコミット前に確認してください。
